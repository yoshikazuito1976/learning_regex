## 📝 正規表現 (Regex) メタ文字チートシート

正規表現でよく使う特殊文字（メタ文字）の意味一覧です。

| メタ文字 | 意味 | 記述例 | マッチする例 |
| :---: | :--- | :--- | :--- |
| `.` | 任意の1文字（改行以外） | `c.t` | cat, cut, cot |
| `^` | 行の先頭 (アンカー) | `^error` | errorで始まる行 |
| `$` | 行の末尾 (アンカー) | `txt$` | txtで終わる行 |
| `*` | 直前の文字の **0回以上** の繰り返し | `ab*c` | ac, abc, abbc |
| `+` | 直前の文字の **1回以上** の繰り返し | `ab+c` | abc, abbc (acは不可) |
| `?` | 直前の文字が **0回または1回** | `colou?r` | color, colour |
| `{n}` | 直前の文字の **n回** の繰り返し | `a{3}` | aaa |
| `[]` | カッコ内の **任意の1文字** (文字クラス) | `[aeiou]` | a, e, i, o, u のどれか |
| `[^]` | カッコ内の文字 **以外** の1文字 | `[^0-9]` | 数字以外の文字 |
| `|` | 左右どちらかのパターン (OR) | `cat|dog` | cat または dog |
| `()` | パターンの **グループ化** | `(ha){2}` | haha |
| `\` | エスケープ (特殊文字を無効化) | `\.txt` | 文字としての ".txt" |

## 🧪 テストの実行方法

このリポジトリには、正規表現をテストするための `regex.txt` が含まれています。
ターミナルで以下のコマンドを実行して、正規表現の挙動を確認します。

### 基本の検索
```bash
# 行の先頭が "error" で始まる行を検索
grep -E '^error' regex.txt

# 1つ以上の数字が含まれる行を検索
grep -E '[0-9]+' regex.txt
```
マッチした部分だけを抽出 (-o オプション)
-o オプションを付けると、行全体ではなくマッチした文字列のみを表示します。

```Bash
# 16進数のカラーコード（例: #FF0000）だけを抽出する
grep -E -o '#[0-9a-fA-F]{6}' regex.txt
```

## 🐍 Pythonでの正規表現 (`re` モジュール)

Pythonで正規表現を使う場合は、標準ライブラリの `re` モジュールをインポートします。
Python特有のポイントとして、正規表現のパターンを書くときはバックスラッシュの挙動をそのまま扱うために **Raw文字列 (`r"パターン"`)** を使うのが一般的です。

### 基本的なメソッド一覧

| メソッド | 説明 | `grep` や他のコマンドとの比較 |
| :--- | :--- | :--- |
| `re.search()` | 文字列全体を走査し、**最初**にマッチした部分を返す | `grep -m 1` に近い |
| `re.match()` | 文字列の**先頭**がマッチするか判定する | `grep '^パターン'` に近い |
| `re.findall()` | マッチした**すべて**の部分をリストで返す | `grep -o` に近い |
| `re.sub()` | マッチした部分を別の文字列に**置換**する | `sed 's/前/後/g'` に近い |

pythonでも同様に正規表現を取り扱えるようになっていけたらいいなと思うよ。
### 💻 Pythonコードの実装例

```python
import re

text = "My phone number is 123-4567 and my zip code is 98765."

# 1. search: 最初の数字の連続を見つける
match = re.search(r'\d+', text)  # \d は [0-9] と同じ意味
if match:
    print(f"見つかった数字: {match.group()}")  # 出力: 123

# 2. findall: すべての数字の連続を見つけてリストにする
all_numbers = re.findall(r'\d+', text)
print(f"すべての数字: {all_numbers}")  # 出力: ['123', '4567', '98765']

# 3. sub: 数字を「***」に伏せ字（置換）する
hidden_text = re.sub(r'\d+', '***', text)
print(f"伏せ字結果: {hidden_text}")
# 出力: My phone number is ***-*** and my zip code is ***.
